# CarScrapper
Simple scrapper of cars from ab.onliner.by, using Scrapy, MongoDB and Flask.

## Run
- run mongo locally, e.g. `mongod --config /usr/local/etc/mongod.conf`
- `cd carscrapper` and `scrapy crawl cars`
- `cd webapp` and `python3 app.py`
